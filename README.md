# NagarWatch 🏙️
### A Geolocation-Based Civic Transparency and Project Monitoring System

---

## 📌 Project Overview
**NagarWatch** is a smart city civic engagement platform designed to enhance transparency and accountability in public development projects. The system allows citizens to view government-funded projects in their local area, receive location-based notifications, report civic issues, and submit geo-tagged photo evidence. Authorities can monitor projects, manage complaints, and respond efficiently through a centralized system.

---

## 🎯 Project Objectives
- Enable transparent monitoring of public development projects  
- Provide location-based visibility using geofencing technology  
- Support citizen participation through issue reporting and evidence submission  
- Improve communication between citizens and local authorities  

---

## ⭐ Key Features
- User registration and area-based access  
- Public project listing with budget and deadline details  
- Geofencing-based notifications for nearby development projects  
- Photo evidence upload with GPS and timestamp  
- Civic issue reporting and status tracking  
- Authority administration panel for monitoring and responses  
- Basic offline support with automatic data synchronization  

---

## 🛠️ Technology Stack
- **Frontend:** Flutter  
- **Backend:** Firebase  
- **Database:** Cloud Firestore  
- **Storage:** Firebase Storage  
- **Maps & Location:** Google Maps API, GPS  
- **Notifications:** Firebase Cloud Messaging
